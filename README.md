## How to Build

Du brauchst Node.js auf deinem System und dazu den `pnpm` Package Manager


To retrieve dependencies

```
pnpm install
```


To build the project

```
pnpm build
```


To work locally run and open `http://localhost:1234/`

```
pnpm start
```
